# An-lise-de-consumo-de-energia
## Descrição
Este projeto apresenta uma solução de automação inteligente para ambientes de trabalho, visando redução de desperdício energético em iluminação e ar-condicionado.

### Objetivos
- Reduzir custos com energia elétrica.
- Implementar monitoramento remoto do consumo.
- Integrar sustentabilidade ao ambiente de trabalho inteligente.

### Dados Utilizados
- Consumo médio mensal: 850 kWh/mês
- Desperdícios identificados:
  - Iluminação: 120 kWh/mês
  - Ar-condicionado: 200 kWh/mês

### Estrutura do Repositório
- `dados/` - Contém os dados utilizados.
- `codigo/` - Scripts em Python para análise e simulação.
- `relatorio/` - Relatório final em PDF.
